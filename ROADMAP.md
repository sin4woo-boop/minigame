# Roadmap

## Vision
- Reach an arcade-quality survivor game with strong combat readability, progression depth, and reliable 60fps performance.

## Milestones

### M1 Core Combat (2-4 weeks)
- [x] Data-driven balance pipeline (`balance.js` + tuning checklist)
- [ ] Weapon roster to 10+ and enemy roster to 8+
- [ ] Wave director v2 with difficulty curves by minute
- [ ] Hit feedback pass (VFX/SFX layering, camera, juice)

### M2 Content & Progression (4-8 weeks)
- [ ] Meta progression (permanent upgrades, unlock tree)
- [ ] Stage themes x3 with map traits/hazards
- [x] Boss system v1 (pattern phases + reward drops)
- [ ] Economy pass (reward rates, risk/reward knobs)

### M3 Production Quality (6-10 weeks)
- [ ] UX pass (tutorial, onboarding, accessibility options)
- [ ] Audio pass (BGM states, SFX mix groups, ducking)
- [ ] Performance pass (object pooling, draw optimization, profiling)
- [ ] Save/replay telemetry foundation

### M4 Launch Readiness (4-6 weeks)
- [ ] QA checklist and regression suite
- [ ] Build/release workflow and versioning
- [ ] Live balance process (patch cadence + metrics)
- [ ] Content backlog for post-launch updates
